# InputText

`InputText` is a React component that renders a textual input (`type: 'text' | 'email' | 'number' | 'password' | 'search' | 'tel' | 'url'`) that is uncontrolled, but whose value is overwritten whenever `props.initialValue` changes. Also, it selects the entire contents of the input whenever the input is focused.
